CinematicFX Plugin - TESTE 3 Delivery
====================================

This package contains the latest build of the CinematicFX Premiere Pro plugin for client testing ("TASTE").

Contents:
- CinematicFX.prm (Premiere Pro plugin binary)
- README_TESTE3.txt (this file)

Instructions:
1. Copy `CinematicFX.prm` to your Premiere Pro MediaCore plugins folder:
   C:\Program Files\Adobe\Common\Plug-ins\7.0\MediaCore\
2. Restart Premiere Pro.
3. Apply the CinematicFX effect from the "Stylize" category.
4. Report any issues or feedback.
