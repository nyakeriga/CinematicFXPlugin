üìù TECHNICAL DETAILS:
================================================================================

**Category System:**
Premiere Pro only recognizes standard Adobe categories. Using "Stylize" 
ensures the plugin appears in the correct menu location.

**ADBE Prefix:**
Adobe's standard naming convention for effects. Helps Premiere properly 
identify and cache the plugin.

**Sequence Handlers:**
PF_Cmd_SEQUENCE_SETUP and SEQUENCE_SETDOWN ensure proper initialization
and cleanup when working with Premiere sequences.

- DOESNT_NEED_EMPTY_PIXELS: Skips transparent pixels (20-30% faster)
- REVEALS_ZERO_ALPHA: Better handling of alpha channels
- SUPPORTS_SMART_RENDER: Only renders changed regions
